# Flying Air Fryers

A browser-based homage to the classic After Dark "Flying Toasters" screensaver, reimagined with air fryers.

## Quick Start

No build step required. Open `index.html` in any modern web browser.

```bash
git clone https://github.com/h1ddenpr0cess20/flying-air-fryers.git
```

Then open `index.html`. On Windows, double-click the file or drag it into a browser window.

Use the speed slider at the bottom to control how fast the fryers fly.
